Hexo安装步骤以及使用中遇到的问题

环境准备:

1、git
2、Node.js
3、Hexo

本地建博客：

hexo init blog
cd blog
nam install

hexo new “title”
在config.xml中编辑博客内容
hexo g
hexo s
hexo d

遇到的问题：

1、ERROR Deployer not found: git

_config.yml 最下方添加

deploy:
type: git
repository: https://github.com/jiamao130/jiamao130.github.io.git
branch: master

deploy的type改成git，然后运行下npm install hexo-deployer-git –save
再hexo g
hexo d

2、当遇到hexo命令无效时，重启终端，使用sudo npm install -g hexo
命令进行安装

3、当出现nvm命令无效时：

git clone https://github.com/creationix/nvm.git ~/.nvm && cd ~/.nvm && git checkout git describe --abbrev=0 --tags

. ~/.nvm/nvm.sh

nvm install 0.12

如果遇到修改完之后没有生效，最好重启下终端或者Mac

4、修改_config.yml时，前面必须要有空格，否则提交会报错
